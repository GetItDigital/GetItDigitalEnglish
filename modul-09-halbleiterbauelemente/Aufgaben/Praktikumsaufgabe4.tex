\subsection{Praktikumsaufgabe 4\label{Aufg32}}
\Aufgabe{

    
    Berechnen Sie die Ausgangsspannung für verschiedene Eingangsspannungen eines Brückengleichrichters.
    
    Gegeben:
    \begin{itemize}
        \item Wechselspannung $\mathrm{U_E}$ mit Amplituden von 0{,}5\,V, 1\,V, 2\,V, 3\,V
        \item Vier idealisierte Dioden (mit 0{,}7\,V Durchlassspannung)
    \end{itemize}
    
    \begin{itemize}
        \item[a)] Bestimmen Sie die Ausgangsspannung für jede Eingangsgröße.
        \item[b)] Erklären Sie, ab welcher Eingangsspannung der Brückengleichrichter funktioniert.
    \end{itemize}
}
